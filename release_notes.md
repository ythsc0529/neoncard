# 🎮 霓虹牌 v1.8.6 更新說明

本次更新帶來全新的 **通知系統**、**設定面板** 與 **背景音樂播放器**，以及戰鬥中的震動回饋，大幅提升遊戲沉浸感！

## 🛠 本次更新重點

### 1. 📢 每日重置通知
*   **每日任務刷新**：每天凌晨 5:00 AM 後首次登入，左下角會彈出 Toast 提示「每日任務已刷新！」
*   **免費抽獎刷新**：同步提示「免費抽獎已刷新！」，確保你不錯過任何免費資源。

### 2. ⚙️ 設定面板（右上角齒輪按鈕）
*   **音樂音量**：即時調整背景音樂音量（0–100%）
*   **音效音量**：預留音效音量調整（後續版本啟用）
*   **戰鬥震動開關**：Toggle Switch 預設開啟，可隨時關閉

### 3. 🎵 背景音樂播放器（右上角音樂按鈕）
*   **自動播放**：App 開啟後自動播放 Neon Shuffle，網頁版在首次互動後開始播放
*   **專屬播放介面**：封面圖、進度條、上一首 / 播放暫停 / 下一首
*   **切歌提示**：左下角彈出「🎵 正在播放：歌曲名稱」Toast 通知
*   **音量同步**：設定面板與播放介面的音量滑塊雙向同步

### 4. 📳 戰鬥震動反饋
*   玩家角色受到傷害或死亡時，手機自動震動
*   震動強度依傷害量調整（40–180ms）
*   同時支援 Web Vibration API（Android Chrome）與 Capacitor Haptics（原生 App）

## 📥 安裝說明
1.  下載下方的 `app-debug.apk`
2.  在手機上開啟檔案，若系統提示「安全性風險」，請選擇「允許安裝來源不明的應用程式」
3.  完成安裝，開啟《霓虹牌》開始你的戰鬥！

## ⚠️ 開發者說明
如果你是開發者並想自行編譯：
*   本專案已遷移至 **Capacitor 6** 架構
*   執行前請確保本地環境具備 **JDK 17** 與 **Android SDK 35**
*   記得在 Firebase 控制台更新你的 **SHA-1** 憑證
