# 🚀 霓虹牌 (Neon Card) - 原生 Android 版正式上線！

我們完成了一次重大的技術飛躍！《霓虹牌》正式從「網頁封裝 (TWA)」進化為**真正的原生架構軟體**。這次更新不僅提升了效能，更解決了過去在手機端最棘手的登入問題。

## 🛠 本次更新重點

### 1. 真正的軟體化架構 (Native Bridge)
*   **本地資源打包**：所有的遊戲圖片與腳本現在直接內建於 App 中。
*   **極速啟動**：不再需要等待網頁載入，開啟 App 即可瞬間進入登入畫面。
*   **更穩定的體驗**：擺脫對手機瀏覽器版本的依賴。

### 2. 原生 Google 登入支援
*   **全新登入流程**：採用 Android 原生帳號選擇器，解決了過去網頁版彈窗被攔截或閃退的問題。
*   **更安全、更快速**：一次點擊即可快速登入遊戲。

### 3. 內置強制更新機制
*   **版本自動偵測**：現在 App 會在啟動時自動檢查新版本。
*   **一鍵下載**：發現新版本時，App 會主動提醒並引導你前往 GitHub 下載，確保每位玩家都在最新的版本進行對戰。

## 📥 安裝說明
1.  下載下方的 `app-debug.apk`。
2.  在手機上開啟檔案，若系統提示「安全性風險」，請選擇「允許安裝來源不明的應用程式」。
3.  完成安裝，開啟《霓虹牌》開始你的戰鬥！

## ⚠️ 開發者說明
如果你是開發者並想自行編譯：
*   本專案已遷移至 **Capacitor 6** 架構。
*   執行前請確保本地環境具備 **JDK 17** 與 **Android SDK 35**。
*   記得在 Firebase 控制台更新你的 **SHA-1** 憑證。
