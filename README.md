# 霓虹牌 (Neon Card Game)

![霓虹牌](https://img.shields.io/badge/Game-Neon%20Card-00f3ff?style=for-the-badge)

這是一款風格獨特的雙人對戰卡牌遊戲，擁有充滿賽博龐克風格的霓虹視覺設計與超過 300 名各具特色的角色。

## 線上遊玩

**快速遊玩：[https://uptocard.netlify.app](https://uptocard.netlify.app)**

## 遊戲特色

*   **多樣化的對戰模式**：
    *   **單人模式**：挑戰多種難度（簡單、困難、地獄）的 AI 電腦，具備聰明的換人與施放技能邏輯。
    *   **雙人對戰**：在同一台裝置上與朋友面對面單挑。
    *   **多人連線模式**：結合 PeerJS 點對點連線技術，只需輸入 4 碼連線代碼即可輕鬆與遠端朋友對決。支援即時表情符號互動！
    *   **劇情模式**：跟隨特別的故事劇情與預設敵人進行充滿挑戰的關卡。
*   **豐富的角色陣容**：超過 300 名角色可供選擇，並帶有專屬技能（包含組合技、資源消耗等）與被動效果。
*   **深度的戰鬥系統**：
    *   戰鬥區與備戰區設計，自由掌控「撤退」時機。
    *   多種異常狀態（暈眩、睡眠、中毒/灼燒、沉默、黑洞吞噬等）。
    *   多元技能機制（斬殺、吸血、反傷、免死、護盾衰減等）。

## 遊戲規則

1.  **卡牌數量**：可選擇 **快速模式 (3張牌)** 或 **經典模式 (7張牌)**。
2.  **開局**：擲硬幣決定先手，各自抽牌後選擇 1 名首發角色進入「戰鬥區」，其餘則進入「備戰區」。
3.  **回合行動**：每回合必須選擇以下其中**一項**行動：
    *   **普攻**：造成等同卡牌 ATK 值的傷害。
    *   **技能**：發動角色專屬技能（施放後需等待冷卻回合 CD，在備戰區也會持續計算冷卻）。
    *   **撤退**：將目前戰鬥區的角色與備戰區的隊友進行交換。
4.  **勝利條件**：成功擊敗對手的所有角色即可獲勝！

## 開發技術

*   **前端**：HTML5, Vanilla JS, CSS3 (搭配發光霓虹特效與 Glassmorphism 玻璃擬物風格)
*   **連線框架**：[PeerJS](https://peerjs.com/) (WebRTC P2P 連線)
*   **部署**：Netlify

## 目錄結構簡介

*   `index.html`：遊戲主選單、模式選擇與連線大廳。
*   `game.html`：核心對戰畫面與邏輯。
*   `tutorial.html` / `strategy.html` / `characters.html`：新手教學、攻略專區及角色圖鑑。
*   `story_menu.html` / `story_runner.html`：劇情模式大廳與劇本演出器。
*   `js/`：包含核心對戰系統 (`battleSystem.js`)、遊戲狀態管理 (`gameState.js`)、連線管理 (`networkManager.js`)，以及所有角色與技能數據 (`data/characters*.js`)。
*   `css/styles.css`：全站共用的霓虹 UI 設計系統。
