## 📖 遊戲簡介

**霓虹牌** 是一款以霓虹美學為主題的雙人對戰卡牌遊戲，純前端實作，無需安裝，開啟瀏覽器即可遊玩。  
玩家從超過 **300 名** 角色中隨機抽取隊伍，利用角色獨特的技能與被動能力擊敗對手。

---

## 🎮 遊戲模式

| 模式 | 說明 |
|------|------|
| 🤖 **單人模式 (PvE)** | 與 AI 對戰，支援**簡單 / 困難 / 地獄**三種難度 |
| 👥 **雙人對戰 (本地 PvP)** | 兩位玩家共用一台裝置進行對戰 |
| 🌐 **多人連線 (Online PvP)** | 透過 4 碼連線代碼建立或加入房間，與遠端玩家對決 |

每種模式均支援：
- **快速模式**：每位玩家擁有 3 張角色牌
- **經典模式**：每位玩家擁有 7 張角色牌

---

## ⚔️ 基本規則

1. **擲硬幣**決定先手玩家
2. 雙方隨機**抽取角色牌**
3. 選擇一張放入**戰鬥區**，其餘置入**備戰區**
4. 每回合選擇一個行動：
   - **普攻**：造成等同 ATK 值的傷害
   - **技能**：使用角色技能（附帶冷卻回合數）
   - **撤退**：將戰鬥區角色與備戰區角色交換
5. 擊敗對手所有角色即可**獲勝**！

> 💡 備戰區的角色技能冷卻也會持續計算，善用撤退時機是致勝關鍵！

## 🌐 線上對戰

v2.1.0 正式實裝多人連線模式，採用 **P2P 點對點連線技術（PeerJS）**：

1. 一位玩家點選「**建立房間**」，取得 4 碼連線代碼
2. 另一位玩家輸入代碼後點選「**加入房間**」
3. 連線成功後自動進入對戰！

> 連線期間具備 **RNG 同步機制**，確保雙方抽牌順序與技能判定完全一致。

## 🚀 如何遊玩

本遊戲為純靜態網頁，只需用瀏覽器開啟 `index.html` 即可。

**本地遊玩：**
```bash
git clone https://github.com/your-username/neoncard.git
cd neoncard
# 用瀏覽器開啟 index.html，或使用 Live Server 等工具
```

**線上遊玩（GitHub Pages）：**  
若已部署至 GitHub Pages，直接前往對應網址即可。

## 🛠️ 技術棧

- **純前端**：HTML5 + CSS3 + JavaScript（無框架）
- **線上連線**：[PeerJS](https://peerjs.com/) v1.5.2（WebRTC P2P）
- **資料儲存**：`localStorage`（遊戲設定與版本紀錄）
